# DM Table Game - Installation Guide

## Quick Install

### Windows:
```cmd
setup.bat
npm start
```

### Mac/Linux:
```bash
chmod +x setup.sh
./setup.sh
npm start
```

## Manual Install

1. Install Node.js from https://nodejs.org/
2. Run: `npm install`
3. Create `.env` file with: `REACT_APP_ANTHROPIC_API_KEY=your_key_here`
4. Run: `npm start`

## Requirements

- Node.js v14+ 
- Anthropic API key (get free credits at console.anthropic.com)
- Internet connection
